During my internship at Gammal Tech, I’ve been learning about the building blocks that allow APIs to communicate with servers.

Today’s topic: HTTP methods.

HTTP methods define the type of action a client wants to perform on a server.

The most common ones are:

• GET → retrieve data
• POST → create new data
• PUT → update existing data
• DELETE → remove data

For example:
A mobile app might send a GET request to retrieve a list of products from a server.

These simple operations form the foundation of many modern API interactions.
