🔹 What is an API?

An API (Application Programming Interface) is a way for different software systems to communicate with each other.

Think of it like a waiter in a restaurant:

You (the user) place an order.

The waiter (the API) takes your request to the kitchen.

The kitchen (the server) prepares the food.

The waiter returns with the result.

In the same way, APIs allow applications to send requests and receive responses from other systems.

For example, when a mobile app shows weather information, it usually gets that data through a weather API.
