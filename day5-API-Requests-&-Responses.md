Today’s topic: API Requests and Responses.

Whenever an application interacts with an API, two things happen:

1️⃣ The client sends a request
2️⃣ The server returns a response

A request typically includes:
• An endpoint (URL)
• An HTTP method (GET, POST, etc.)
• Headers
• Sometimes a request body

The server processes the request and sends back a response containing:
• A status code
• Data (often in JSON format)

This request-response model is the core mechanism that allows applications to exchange information through APIs.
