🔹 What is a REST API?

REST stands for Representational State Transfer. It’s an architectural style that allows systems to communicate using standard HTTP requests.

REST APIs are popular because they are simple, scalable, and easy for developers to understand.

Some characteristics of REST APIs include:
• They use standard HTTP methods
• They are stateless (each request contains all needed information)
• They often return data in JSON format

Because of their simplicity and flexibility, REST APIs power many modern web and mobile applications.
