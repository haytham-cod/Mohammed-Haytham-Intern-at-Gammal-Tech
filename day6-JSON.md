A format you will see almost everywhere in APIs is JSON.

🔹 What is JSON?

JSON stands for JavaScript Object Notation. It is a lightweight format used to store and exchange data.

It’s popular because it’s:
• Easy for humans to read
• Easy for machines to parse
• Compatible with many programming languages

Example of JSON data:

{
"name": "Mohammed",
"role": "Intern",
"company": "Gammal Tech"
}

Because of its simplicity and flexibility, JSON has become the standard format used by most APIs today.
